Youtube playlist jquery and php Gdata Api html5 installation for wordpress

Requires PHP 5 and Curl or url fopen. It is coded using PHP 5 OOP and wordpress 3.x

Unzip the file and upload the dir youtubeplaylist to
wordpress/wp-content/plugins/

Go to your admin page and click on plugins - Youtube SEO Playlist and click activate

Now go to Settings and click on Youtube Playlist
Here you will see the shortcodes to use for your pages and posts and all kind of settings.

For all the settings examples go to http://www.cfcms.nl/youtube/

Good luck and happy viewing
