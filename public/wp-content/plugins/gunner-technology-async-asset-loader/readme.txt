A plugin leverages Modernizr to load JavaScript files asynchronously 

== Description ==

Just drop this plugin in, activate it and you're all set.

Your WordPress site will automatically start to take advantage of the asynchronous loading of JavaScript files provided by the awesome JavaScript library, [Modernizr](http://www.modernizr.com/).

Read more about this awesome feature in the [Modernizr Docs](http://www.modernizr.com/docs/#load).

For questions, comments and feature suggestions, please go to our post: [Modernizr WordPress Plugin](http://gunnertech.com/2012/02/modernizr-wordpress-plugin).



Requires WordPress 3.0 and PHP 5. 


== Installation ==

1. Upload the files to the `/wp-content/plugins/gt-authorship/` directory
1. Activate the "Gunner Technology Async Asset Loader" plugin through the 'Plugins' menu in WordPress
