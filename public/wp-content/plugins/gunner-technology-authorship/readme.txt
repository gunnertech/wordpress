== Description ==

The Gunner Technology Authorship plugin allows WordPress authors to add their social credentials to their articles and author pages and creates the authorship meta connection with Google+.

[This article by Yoast](http://yoast.com/wordpress-rel-author-rel-me/) inspired the plugin.

But it adds the ability to Twitter Follow, Facebook Like, Facebook Subscribe and Tungle Schedule.

For any more questions, please go to Gunner Technology's post: [Authorship WordPress Plugin by Gunner Technology](http://gunnertech.com/2012/02/authorship-wordpress-plugin-by-gunner-technology/) and leave a comment.

[youtube  http://www.youtube.com/watch?v=nY2EC4Doebs]

Requires WordPress 3.0 and PHP 5. 


== Installation ==

1. Upload the files to the `/wp-content/plugins/gt-authorship/` directory
1. Activate the "Gunner Technology Authorship" plugin through the 'Plugins' menu in WordPress
