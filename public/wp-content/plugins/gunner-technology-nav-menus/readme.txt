A plugin that allows authors to create menu locations and add menus via widgets.

== Description ==

The Gunner Technology Dynamic Nav plugin allows WordPress admins to create an unlimited number of Theme Locations for menus and then populate these locations with menus via widgets.

For questions, comments and feature suggestions, please go to our post: [Dynamic Menus WordPress Plugin by Gunner Technology](http://gunnertech.com/2012/02/dynamic-menus-wordpress-plugin) and please check out the video below.

[youtube http://www.youtube.com/watch?v=V3wv4p_f7jA]

Requires WordPress 3.0 and PHP 5.3. 


== Installation ==

1. Upload the files to the `/wp-content/plugins/gt-dynamic-menus/` directory
1. Activate the "Gunner Technology Dynamic Menus" plugin through the 'Plugins' menu in WordPress
